2. 參考 http://notepad.yehyeh.net/Content/DS/CH04/3.php 的 Insert Node

   撰寫一支程式，使用者每輸入一個整數，便將它按照由小到大的順序插入到Linked List中，

   然後將Linked List節點裡的值一一display到螢幕上。

   ```
   例如：

   Input an integer? 3

   3

   Input an integer? 2

   2  3

   Input an integer? 5

   Input an integer? 1

   ```
